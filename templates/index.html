Why this matters

Methodology (Linear Regression)

1) Data collection & understanding

Source: Algerian forest fire dataset (244 observations across two regions). Target variable: FWI (continuous). Predictors: weather and fire indices such as Temperature, RH, Ws, Rain, FFMC, DMC, ISI, plus Classes and Region.

2) Preprocessing & cleaning

Handled missing values by removing incomplete rows. Detected outliers via distribution checks and descriptive stats. Encoded categorical values (Classes/Region) into numeric form. Standardized features using StandardScaler to keep all inputs on a comparable scale.

3) Exploratory data analysis

Built correlation heatmaps to understand relationships and spot multicollinearity. Used scatter plots and distributions to verify linear trends and feature behavior.

4) Model development & assumptions

Split data into training and testing sets to avoid overfitting. Used multiple linear regression and checked assumptions: linearity, homoscedasticity, independence, normality of errors, and low multicollinearity.

5) Training & evaluation

Trained via Ordinary Least Squares (OLS). Evaluated with $R^2$ / adjusted $R^2$, RMSE, and MAE. Reviewed residual plots to confirm random error patterns and model validity.

6) Tools & libraries

Python with Pandas, NumPy, Scikit-learn, Matplotlib, and Seaborn.
